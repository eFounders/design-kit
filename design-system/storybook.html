Storybook

The whole design system, for real. Play with the panel at the top: change the brand color (hex), switch to dark, adjust the radius, the density, the font and the icon library. Everything updates live. This is the tool to showcase the DS and explain the rules.

Colors
Every color comes from tokens.css, in 2 tiers: primitives → semantic tokens. Always use the semantic token (e.g. --border), never a primitive. Feedback colors serve ONLY their meaning.

7 sizes, whole pixels. The Density control rescales the whole ramp (14 dense ↔ 16 comfortable). Live px shown per step. Hierarchy comes from size and weight, not color.

Spacing — 4px grid
Multiples of 4 only. Spacing creates the structure (related items close, groups apart), not borders.

Badges & tags
Badges = status, semantic meaning only (success, warning…). Tags = decorative labels (categories, custom statuses) with a free color palette. Different jobs, different tokens. For an action → Button, not a badge.
